Synthetic Data

합성/재현 데이터
|
모형 및 알고리즘을 이용해 원본 데이터와 유사한 구조를 가지도록 만들어진 가상(simulated) 데이터입니다. 합성 데이터를 사용해 원본 데이터 유출 우려를 줄일 수 있고 대량의 데이터를 빠르게 생성할 수 있습니다.

Use Cases

활용 사례
| 개인정보 보호
원본 데이터가 아닌 합성 데이터를 사용함으로써 데이터 유출 우려를 줄일 수 있습니다.
| 모델 개발 및 테스트
합성 데이터를 사용해 원본 데이터에 영향을 미치지 않고 모델을 개발하고 테스트할 수 있습니다.
| 데이터 공유
원본 데이터를 공유할 필요 없이 합성 데이터를 공유해 데이터 공유 문제를 해결할 수 있습니다.
| 데이터 분석 및 시각화
합성 데이터를 사용해 원본 데이터의 보안 문제를 우려할 필요 없이 분석 결과를 공유할 수 있습니다.

Fields

적용 가능 분야
| 의료 분야
의료 데이터를 합성해 임상시험, 인공지능 알고리즘 학습에 활용 가능
| 금융 분야
거래 데이터를 합성해 모의 거래 시뮬레이션 가능
| 보안 분야
해킹 등 보안 위협 시나리오를 합성해 보안 대응 전략 개발에 활용 가능
| 자율주행 분야
다양한 운전 상황을 합성해 자율주행 알고리즘 개발에 활용 가능
| 게임 분야
유저 데이터를 합성해 게임 개발 및 테스트 가능
